# Automation Testing Project

هذا المشروع هو اختبار **Automation Testing** باستخدام **Selenium** و **Python**. الهدف من المشروع هو أتمتة عملية الدخول إلى موقع ويب، والتسجيل باستخدام البيانات المطلوبة، ثم إجراء تسجيل الدخول بشكل تلقائي.

## محتويات المشروع

- **Selenium**: مكتبة Python التي تم استخدامها لأتمتة التفاعلات مع المتصفح.
- **WebDriver**: يتم استخدام Chrome WebDriver للتفاعل مع المتصفح.
- **Python**: لغة البرمجة التي تم استخدامها لكتابة السكربتات.

## المتطلبات

- Python 3.x
- Selenium
- WebDriver (مثل ChromeDriver)
